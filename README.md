# Nompose

```
nompose 0.1.0
Convert docker-compose files to nomad files.

USAGE:
    nompose [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -i <docker-compose-file>        Compose File: docker-compose file name [default: docker-compose.yaml]
    -o, --output <output>            [default: stdout]
```
